В IPv4-сети есть локальный и публичный IP-адреса.

Локальный IP-адрес предназначен для идентификации вашего устройства в локальной сети и не может быть использован для доступа в Интернет. Используйте локальный IP-адрес для подключения к вашему серверу.

Публичный IP-адрес может быть доступен через Интернет. Используйте его, чтобы пригласить других людей вне вашей сети. (Сначала требуется настройка переадресации портов)

Вы не обязаны использовать эти IP-адреса для подключения к вашему серверу. Сервер будет автоматически отображен во вкладке "друзья" в игре.

Эти IP-адреса предназначены для использования в разработке или тестировании.
